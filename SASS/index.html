Mixing e Includes

Serve para evitar várias repetições, onde os elementos sempre usam os mesmos parâmetros

@mixin container-flex($flow: row, $align: stretch, $justify: start, $gap: 1rem) {
    display: flex;
    flex-flow: $flow;
    align-items: $align;
    justify-content: $justify;
    gap: $gap;
}

@mixin textos($size: 1.6rem, $weigth: 100, $height: 1.4) {
    font-size: $size;
    font-weight: $weigth;
    line-height: $height;
}

Onde usaria por exemplo, display, flex-flow, font-size etc. você usa

@include [nome do mixin({argumentos})]

Para usar argumentos específicos, deve declarar no uso, e os outros serão ignorados, usando o padrão. Exemplo:

@include  textos($size: 2rem)

@include container-flex($flow: column, $gap: 1.6rem)

Mixing e Includes - Responsivo

Serve para evitar várias repetições, e deixar a página responsiva com mixin

@mixin responsive($tamanho) {
    @if $tamanho == small {
        @media (max-width: 27.5em) {
            @content;
        }
    } @else if $tamanho == medium {
        @media (max-width: 48em) {
            @content;
        }
    } @else if $tamanho == big {
        @media (max-width: 90em) {
            @content;
        }
    } @else {
        @media (max-width: 110em) {
            @content;
        }
    }
}

Você define o tamanho da media querie dentro do elemento

.classe-elemento{
    font-size: 2rem;

    //exemplo de querie para mobile
    @include responsive(small){
        font-size: 1.6rem;
    } 
}

Placeholder e Extend

para usar um placeholder usa o %, no elemento uso o @extend %placeholder, como por exemplo para links/botão

%botao-lv {
    align-self: center;
    color: #e7b4b4;
    font-weight: 500;
    background-color: #303030;
    border-radius: 20rem;
    padding: 1.02rem 2.5rem;
    text-decoration: none;
    transition: all ease-in 150ms;
}
%botao-ha {
    align-self: center;
    font-weight: 500;
    color: #8b0000;
    background-color: #30303013;
}

Exemplo de uso:

&__btn:link,
&__btn:visited {
    @extend %botao-lv;
}

&__btn:hover,
&__btn:active {
    @extend %botao-ha;
}

Nesting SASS

.caixa__main{
    &--h1{}
}

o & copia toda a classe pai ex.: caixa__main--h1 o & repete o caixa__main

Funções em SASS

Para criar funções em sass

@function multiplicar($a, $b) {
    @return $a * $b;
}
@function dividir($a, $b) {
    @return $a / $b;
}
